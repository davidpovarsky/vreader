## Observations

- **Two ViewModels for one surface â€” by design.** General chat and
  book chat use the same `AIChatViewModel` class with a nil-or-set
  `bookFingerprint`. This avoided a copy-paste fork at the cost of
  one extra branch in `sendMessage` (prepend system message? yes /
  no). The bifurcation tests pin the branch decision.
- **Library toolbar gating is duplicated by design.** The entry
  point checks feature-flag + API-key + consent; the
  `AIChatViewModel` re-checks the same gates on `sendMessage`.
  Belt-and-suspenders â€” if the UI ever shows the entry point in
  error, the ViewModel still refuses to send. Tests pin both
  layers.
- **Sliding-window history caps memory + token cost.** The
  ViewModel keeps the last N turns when prompting the provider;
  beyond that, oldest turns drop. Test `slidingWindowDropsOldMessages`
  pins the eviction.
- **Unicode / CJK preservation is non-trivial when prompts get
  re-encoded for HTTP.** The test sends a CJK string and asserts
  the assistant turn round-trips it; if the provider client
  ever started escaping, this catches it.
- **AIService gate order: flag â†’ consent â†’ API key.** This order
  matters because the error message tells the user what to do
  next. "Enable AI in Settings" is a different action from
  "Accept the privacy notice" is a different action from "Add an
  API key". Wrong order â‡’ wrong user-facing message â‡’ user
  follows the wrong recovery path.
- **Cache key determinism = no duplicate API spend.** Action type +
  prompt version + params hash to the cache key. Bumping the
  prompt version is the project's mechanism for invalidating
  cached responses when the system prompt changes. Without
  prompt-version-in-cache-key, a prompt-engineering change would
  silently re-use stale responses.
- **Live feature-flag toggle.** `featureFlagToggleLiveReflectsInService`
  catches the bug class where the service caches the flag at
  init time. After landing, the user toggling AI off in Settings
  immediately denies new requests; without this test, the bug
  could regress silently because the surface is rare.
