## Bug discovered (filed, NOT fixed)

**Bug #155 / GH #451** — Library collection filter selection has no effect: tapping a collection in the sidebar dismisses the sheet but the library view still shows ALL books, not just the collection's members.

Root cause (code-read): `LibraryView.gridView` (`vreader/Views/LibraryView.swift:396`) and `LibraryView.listView` (line 422) iterate `viewModel.books` directly via `ForEach(viewModel.books)`, never consulting `activeFilter` (line 52) when building the rendered list. `CollectionSidebar.swift:65-67` correctly sets `activeFilter = .collection(name)` but the wiring stops there.

Per cron rules, NOT fixed in this verification iteration; tracked as bug #155 with full root-cause + fix sketch in `docs/bugs.md`.
